# 🦂 Scorpio Search Engine

Scorpio Search Engine adalah mini search engine berbasis Node.js yang memanfaatkan Wikipedia API sebagai sumber data pencarian. Proyek ini dibuat untuk keperluan pembelajaran, portofolio, dan tugas mahasiswa Informatika.

---

## 🚀 Features

- 🔍 Search Engine
- 💡 Autocomplete Suggestion
- 📜 Search History (LocalStorage)
- 📄 Pagination
- 🌙 Dark Mode
- 📊 Search Statistics
- ⭐ TF-IDF Relevance Ranking
- 🔗 Open Result in New Tab
- 📱 Responsive Design
- ⚡ Fast Search

---

## 🛠 Built With

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Vercel Serverless Function
- Wikipedia API

## ⚙️ Installation

Clone repository

```bash
git clone https://github.com/F3ndi-C4ndra/scorpio.git
```

Install dependency

```bash
npm install
```

Run project

```bash
npm start
```
